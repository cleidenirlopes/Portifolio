Connecting Lisbon’s Public Transport with Communities.

Lisbon Urban Mobility Insights is an initiative that connects real-time data from Lisbon’s bus network with local communities and decision-makers to improve service quality, accessibility, and engagement. By integrating live transport data and advanced geospatial analysis, the project brings visibility to public transit operations and empowers communities with meaningful insights.

Description of the Project

The Lisbon Urban Mobility Insights project was developed to demonstrate the power of real-time analytics in public transport. Using Python, SQL, and Power BI, the project provides dashboards that show the operational status of Lisbon's bus lines minute by minute. The focus is to offer a transparent and accessible view of how the transport system functions in real time, supporting planners, citizens, and researchers alike.

Steps Taken in the Project:

Data Integration:

  • Real-Time API Collection: Public data sources were accessed to retrieve live data from Lisbon's bus and tram systems, including vehicle locations, speed, operational status, and line IDs.
  • Operational Status Logic: Vehicles were categorized as "active," "stopped," "arriving," or "completed" using timestamped GPS coordinates and status flags.
  • Line Categorization: Using patterns in the line IDs, trams and buses were separated, and each was analyzed for line usage per hour and per region.

Core Concept – Interactive Dashboard Metrics:

The dashboard allows users to monitor operational metrics such as total buses per hour, buses in operation per moment, buses completing their trips, and vehicles stopped in real time. Each card in the interface is interactive, filtering the data in maps and charts like BusLineDistrict, BusPerLine, and BusLineShare when selected.

Impact Model:

  • Public Planners: Make informed decisions on fleet distribution and frequency adjustments.
  • Residents: Gain transparency on public transport operations in their neighborhood.
  • Students & Researchers: Use the real-time dataset for urban mobility studies and machine learning models.

Target Locations and Use Cases:

The dashboard covers the entire bus network operated by Carris, analyzing all stops, districts, and municipalities within Lisbon. It is designed for broad use cases such as:

  • Urban planning and transportation policy decision-making.
  • Commuter pattern analysis across multiple districts.
  • Service optimization for public transport authorities.

Users can explore movement density, coverage by district, and line saturation in peak hours.

Unique Value Proposition:

  • For Communities: Understand how many vehicles are circulating in their area in real time.
  • For Technicians: Detect issues such as bottlenecks, delays, or gaps in service.
  • For Decision Makers: Prioritize infrastructure investments and line restructuring.

Way Forward:

  • Expand Modal Coverage: Integrate Lisbon Metro and ferry services.
  • Historical Trends: Store and visualize historical data to analyze long-term patterns.
  • User Feedback: Allow users to suggest adjustments to improve public service based on dashboard findings.

Conclusion:

This real-time transport dashboard turns data into decisions. It reveals Lisbon’s transport performance minute by minute and bridges the gap between service providers and the public. With continued development, it will grow into a foundational tool for data-driven urban planning.
